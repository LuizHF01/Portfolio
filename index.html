Sobre mim

Sou um desenvolvedor Front End. júnior fascinado por criar interfaces atraentes e funcionais para a web. Embora eu ainda não tenha experiência profissional na área, tenho conhecimentos sólidos em HTML, CSS e Java Script e estou constantemente buscando maneiras de me atualizar e aprimorar minhas habilidades. Estou em busca de oportunidades para aplicar meus conhecimentos e habilidades em um ambiente de trabalho profissional e aprender com profissionais experientes.

O profissional desenvolvedor frontend é o responsável por criar através de códigos, o design de um site ou interface. O Frontend implementa o design através de códigos, como HTML, CSS, Java Script e algumas bibliotecas e Frameworks.
O profissional desenvolvedor backend trabalha na parte de “trás” da aplicação. Ele é o responsável, pela implementação da regra de negócio. Em uma aplicação web, este desenvolvedor, quando focado, não toca na parte visual da aplicação.
O DBA (database administrator), sigla em inglês para Administrador de Banco de Dados, é um profissional da área responsável pela criação, instalação, monitoramento, reparos e análise de estruturas de um banco de dados.
